# AgentCribs Community

AgentCribs is a working community spun out of PWV founder sessions where builders share real AI development workflows, demo artifacts, and reusable resources.

The published site lives in `docs/` and is intended for GitHub Pages. Installable bundles, transcripts, and other supporting materials live in `resources/`.
